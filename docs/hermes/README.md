# Hermes
> Asistente virtual inteligente para negocios

## Descripción

Hermes es un asistente virtual impulsado por inteligencia artificial diseñado para revolucionar la atención al cliente en negocios. Más que un simple chatbot, Hermes es una solución completa que aprende y se adapta a las necesidades específicas de cada empresa, ofreciendo una experiencia de atención al cliente excepcional las 24 horas del día, los 7 días de la semana.

El sistema permite a las empresas automatizar respuestas frecuentes, gestionar consultas de clientes de manera inteligente y proporcionar información precisa en tiempo real, todo mientras mantiene un tono conversacional natural y profesional. Hermes se integra fácilmente en múltiples canales de comunicación, incluyendo sitios web, WhatsApp y sistemas de kioscos.

Como producto estrella de Zenova Tech, Hermes representa la visión de la empresa sobre cómo la inteligencia artificial puede transformar la relación entre negocios y clientes, mejorando la satisfacción del cliente mientras reduce costos operativos.

## Stack Tecnológico

### Frontend
- Framework: Astro.js 4.x
- SSR: Server-side rendering con Node adapter
- Styling: TailwindCSS
- Formularios: Validación nativa con reCAPTCHA v3
- Analytics: Google Tag Manager, Google Analytics

### Integraciones
- Email: Nodemailer con Gmail SMTP
- reCAPTCHA: Google reCAPTCHA v3 para seguridad
- WhatsApp: Integración directa para chat
- Meta: Integración con plataforma Meta

### Infraestructura
- Deployment: Coolify con Docker
- Dominio: stovia.uno
- SSL: Certificados automáticos
- Lenguaje: Español (Argentina)
